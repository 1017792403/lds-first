# 进度日志 — Project_01: 股票分析工具优化

## 项目目标

将 12 个零散的股票量化分析脚本重构为规范化的 Python 包，并提供 Web 界面和自动调度功能。

---

## 已完成功能

### 核心功能
- [x] 模块化重构：12 个脚本 → 18 个模块的 `stock_analyzer` 包
- [x] 三种选股策略：Basic（经典评分）、V2（过热修正）、V3（事件+技术+人性）
- [x] 多线程并发行情拉取（~3s vs 原始 ~30s）
- [x] SQLite 数据缓存（行业 7 天、K 线 1 天、行情 5 分钟）
- [x] 技术指标计算：MACD、KDJ、量比、斜率、量价背离

### Web 界面
- [x] Flask Web App（端口 8765）
- [x] 仪表盘：一键运行三种策略 + 得分图表 + 行业分布
- [x] 可视化报告：Chart.js 交互式 HTML 报告
- [x] 设置页面：浏览器中动态修改选股逻辑参数
- [x] 复盘页面：历史选股记录和胜率统计
- [x] 回测页面：basic vs v2 策略对比

### 自动部署
- [x] GitHub Actions 定时任务（08:00/17:00 北京时间）
- [x] 本地自动调度器（服务器运行时到点自动跑）
- [x] 选股结果自动提交到 GitHub 仓库

## 技术决策

| 决策 | 选择 | 理由 |
|---|---|---|
| 端口 | 8765 | 避免与常见端口冲突 |
| 分类数据 | `data/sw.json` | Excel 在服务器环境不可用 |
| 推送认证 | `secrets.GITHUB_TOKEN` | Actions 内置，无需额外配置 |
| 复盘记录 | 仅自动调度写入 | 避免手动点击污染数据 |
